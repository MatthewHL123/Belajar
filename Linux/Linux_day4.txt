Hari 4: Manipulasi File dan Direktori
Perintah: cp, mv, rm, mkdir, touch. Latihan: Buat direktori, salin file, pindahkan file, dan hapus file.

cp (Copy)
Menyalin file atau direktori.
Salin file:
cp file.txt salinan_file.txt

Salin direktori (rekursif):
cp -r folder_asli folder_salinan

mv (Move/Rename)
Memindahkan file/direktori atau mengganti nama.
Pindahkan file ke direktori lain:

mv file.txt direktori_tujuan/

Ganti nama file:

mv file.txt file_baru.txt

rm (Remove)
Menghapus file atau direktori.
Hapus file:

rm file.txt

Hapus direktori (rekursif):
rm -r folder

mkdir (Make Directory)
Membuat direktori baru.
Contoh:
mkdir folder_baru

touch
Membuat file kosong baru.
Contoh:
touch file_baru.txt
